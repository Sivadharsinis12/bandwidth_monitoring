🚀 NetBandwidth – Smart Bandwidth Monitoring & Control System

A real-time network monitoring system that captures live traffic from connected devices and visualizes bandwidth usage through an interactive dashboard.

Built using FastAPI, WebSockets, Scapy, HTML, CSS, and JavaScript.

📌 Features

📡 Real-time packet capture using Scapy

📊 Live bandwidth graph (WebSocket-based updates)

🖥️ Multi-device detection on local network

⚡ Upload & Download monitoring

🚨 High usage alerts detection

📁 Historical usage tracking

🎯 Device management dashboard

🔄 Real-time data streaming

💡 Student-level implementation of IDS-like monitoring
